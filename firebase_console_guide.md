# Firebase Console Notifications Guide

## Како да испраќаш нотификации од Firebase конзола

### 1. Отвори Firebase Console
- Иди на: https://console.firebase.google.com
- Избери го твојот проект: `meal-lab2-project`

### 2. Иди на Cloud Messaging
- Од левото мени избери "Messaging"
- Кликни на "Send your first message"

### 3. Испрати тест нотификација
- **Notification title**: "Recipe Alert!"
- **Notification text**: "Try a new delicious recipe today!"
- **Target**: Select app → com.example.lab_2 (Android)
- Кликни "Send test message"

### 4. Најди FCM Token
Кога ќе ја стартираш апликацијата, во конзолата ќе се прикаже:
```
FCM Token: [долг низ од букви и бројки]
```

Овој токен го користи за тестирање на одредени уреди.

### 5. Тестирање
- Испрати нотификација од конзолата
- Нотификацијата ќе се прикаже на телефонот
- Ако апликацијата е затворена, нотификацијата ќе се прикаже во системот
- Ако апликацијата е отворена, ќе се прикаже порака во конзолата

## Што може да правиш:
✅ Испраќај нотификации од Firebase Console
✅ Нотификациите работат и кога апликацијата е затворена
✅ Можеш да таргетираш сите уреди или специфичен токен
✅ Можеш да додаваш слики и копчиња во нотификациите
